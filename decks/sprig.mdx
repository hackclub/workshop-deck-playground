# Randomly move the enemy, Orpheus

Every time the player moves, Orpheus randomly moves.

```js
getFirst(orpheus).y += Math.FLOOR(Math.random() * 5) - 2;
```

ü´µ Add this in your `onInput()` functions.

# Check if the player and Orpheus overlap

Put this in your `afterInput()` function.

```js
if (
  getFirst(player).x == getFirst(orpheus).x &&
  getFirst(player).y == getFirst(orpheus).y
) {
  // something here!
}
```

---

# Add some "you win" text

Put this inside the `if` statement you just wrote.

```js
addText("you win!", {
  x: 10,
  y: 4,
  color: color`3`,
});
```
